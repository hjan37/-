# 한국형 여론조사기관 등급제 분석 데이터

## 📌 개요

본 리포지토리는 다음 논문에서 사용된 분석 데이터와 산출 결과를 수록하고 있습니다.

> **미국 538 여론조사기관 등급 모델의 한국 적용 가능성 연구**
> — 제20·21대 대통령선거 데이터를 중심으로

본 연구는 미국 FiveThirtyEight(538)의 여론조사기관 평가 방법론을 한국적 맥락에 맞게 재구성하여, 제20대(2022) 및 제21대(2025) 대통령선거 여론조사 데이터를 대상으로 기관별 등급을 산출하는 시뮬레이션을 수행한 것입니다.

| 항목 | 내용 |
|------|------|
| 분석 대상 | 제20대·제21대 대통령선거 여론조사 |

## 📐 주요 변수 설명

| `ActualGap` | 실제 선거결과 득표율 격차 |

| `AbsError` | 원시 오차 (Error의 절댓값) |
| `ExcessError` | 초과 오차 (AbsError − ExpectedError) |
| `APM_row` | 개별 조사의 정규화 절대오차 (AbsError / ExpectedError) |
| `Bias_row` | 개별 조사의 편향값 |

### `3.Summary` 시트 변수

| 변수명 | 설명 |
|--------|------|
| `Pollster` | 여론조사기관명 |
| `N_polls` | 분석에 포함된 조사 건수 |
| `MAWE` | 가중평균 절대오차 (Mean Absolute Weighted Error) |
| `APM` | 조정 성과 지표 (Adjusted Performance Metric) — 값이 낮을수록 정확 |
| `SPM` | 표준 성과 지표 (Standard Performance Metric) — 값이 낮을수록 일관적 |

| `PPM` | 성과 정밀도 지표: √(APM² + SPM²) — 값이 낮을수록 종합 성과 우수 |
| `BiasPenaltyFactor` | 편향 크기에 따른 페널티 계수 |
| `PPM_BiasAbs` | 편향 페널티 적용 후 PPM |
| `Rank` | 순위 (PPM 기준, 1위가 가장 우수) |
| `Grade` | 등급 (A/B/C, 삼분위 기준 상대 평가) |

## 🔧 핵심 산출 공식

```
TimeWeight   = exp(−Days / τ),  τ = 45
SizeWeight   = √(S / 1000)
FinalWeight  = TimeWeight × SizeWeight
ExpectedError = 0.98 / √n
ExcessError  = |PollGap − ActualGap| − ExpectedError
APM_j        = Σ(w_i · APM_row_i) / Σw_i
SPM_j        = √[Σw_i · (APM_row_i − APM_j)² / Σw_i]
PPM_j        = √(APM_j² + SPM_j²)
```

## 📈 등급 구분 기준

논문 본문에서는 **삼분위수(k=3)** 기준을 적용하였습니다.

| 등급 | 기준 | 기관 수 |
|------|------|---------|
| A | PPM ≤ 2.80 (하위 33%) | 13개 |
| B | 2.80 < PPM ≤ 9.60 (중위 33%) | 13개 |
| C | PPM > 9.60 (상위 33%) | 14개 |

감응도 분석(k=3~6)에서 등급이 변동한 기관은 1개(케이스탯리서치: A→B)로, 등급 구분의 구조적 패턴은 대체로 안정적으로 유지되었습니다.
